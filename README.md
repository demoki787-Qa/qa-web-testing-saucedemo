# QA Web Testing Project - SauceDemo

Projeto de testes manuais realizado no sistema web SauceDemo com o objetivo de praticar e demonstrar habilidades em Quality Assurance (QA).

## Objetivo

Realizar testes funcionais em um sistema web de e-commerce para validar funcionalidades, identificar bugs e documentar resultados de teste.

## Sistema Testado

SauceDemo é uma aplicação web que simula um e-commerce com funcionalidades de login, listagem de produtos, carrinho de compras e checkout.

Site testado:
https://www.saucedemo.com

## Tipos de Teste Realizados

- Testes funcionais
- Testes de login
- Testes de fluxo de compra
- Testes negativos
- Validação de mensagens de erro

## Cenários Testados

- Login com credenciais válidas
- Login com credenciais inválidas
- Adicionar produto ao carrinho
- Visualizar carrinho
- Processo de checkout
- Validação de campos obrigatórios

## Estrutura do Projeto

TestCases/
- login_tests.md
- checkout_tests.md

BugReports/
- bug_report_01.md

Evidence/
- screenshots dos testes realizados

## Ferramentas Utilizadas

- Navegador Chrome
- Testes manuais
- Documentação de bugs

## Resultado

Durante os testes foram executados diversos cenários de uso do sistema e identificadas possíveis falhas de validação e comportamento inesperado.
